
# Исследование исторических данных по компьютерным играм.

-------------------------------------------
 

### Краткое описание задачи

Были предоставлены исторические данные из открытых источников о продажах игр, оценки пользователей и экспертов, жанры и платформы (например, Xbox или PlayStation). Необходимо выявить определяющие успешность игры закономерности. 

-------------------------------------------

### В работе использованы:

- библиотеки Pandas, Seaborn, MatplotLib, SciPy, NumPy

-------------------------------------------

### Описание работы

1. Проведена предобработка данных, удалены все пропущенные значения.
2. Проведен исследовательский анализ данных: 
      - выявлены закономерности популярности игровых платформ
      - выбран временной промежут данных, на основе которых можно предсказать, какие игровые платформы, жанры игр и т.д. буду популярны в следующем году
      - составлен портрет среднего пользователя стран Европы, Азии и Северной Америки
      - была проедена проверка статистических гипотез
